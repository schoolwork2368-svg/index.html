1. What is JavaScript
JavaScript (abbreviated as JS) is a crucial part to making webpages. JavaScript, unlike HTML or CSS, allows for a page to have
 interactivity in it (buttons, commands).
			    

				
			         
2. What are var and let
var is a statement and is similar to let in that it declares a function when a specific thing has happened. var will activate 
when something triggers it using if. var will than update the trigger and create a new value.

let is a declaration and is similar to var in that it declares a function when a specific thing has happened. let will activate 
when something triggers it using if. let will than run a new string of code without updating the trigger.



3. What are Data types
Data types are different ways that a programming language can determine the value of an imput and produce andexpected output.



  • Null - This an object type which make something not exist or appear.
    • 

  • Undefined - This is an undefined type which makes something undefined or unchosen.
    • 

  • Boolean - This is a boolean which allows true or false as inputs to se things as true or false. True and false are reserved keywords 
          meaning you can't use these as variables.
    • 

  • Number - This is a number type which can set a variable to a number.
    • 

  • BigInt - This is a number type which can keep precision with masive numbers over 2^53-1 (which is 9,007,199,254,740,991), known as Number.MAX_SAFE_INTEGER. The normal Number can lose precision after a certain point so you use BigInt.
    • 

  • String - This is a String type which will be used over a string of code e.g. const result = data.filter(n => n % 3 !== 0).map(x => x ** 2);. It is mainly an identifier for a string.
